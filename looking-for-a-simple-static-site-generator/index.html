Super easy to configure:

  • Create a Github account
  • Enable Github pages
  • Download the lightweight Publii site tool
  • Create a test post, push to Github via the built-in Git publisher
  • ?
  • Profit!

Did I mention it's open source and can import existing Wordpress sites? :)
